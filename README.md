# ✅ Groot Todo - Modern React Context API Task Manager

<div align="center">
  <h3><b>Context API State Management • Theme Toggles • Fluid Layouts</b></h3>
  <p>A Groot-themed React application exploring Context API state structures and smooth animations.</p>
</div>

---

## 🌿 Features
- **Global State:** Uses React Context API (`useContext`) to manage task lists and themes globally, preventing prop-drilling.
- **Theme Switcher:** Beautiful dark and light modes styled with CSS custom variables.
- **Task Management:** Create, delete, and check off tasks.
- **Search & Filter:** Instantly filter tasks by search term or completion state.
- **Fluid Layouts:** Responsive design that fits perfectly on mobile and desktop screens.

---

## 🛠️ Tech Stack
- **Frontend:** React.js, JavaScript, HTML5
- **Styling:** CSS Modules with custom variables (Light/Dark themes)
- **Deployment scripts:** Configured for Netlify, Vercel, and GitHub Pages

---

## 🚀 Getting Started

1. **Clone & Install:**
   ```bash
   git clone https://github.com/the-sage-00/groot-todo-learning-project-.git
   cd groot-todo-learning-project-
   npm install
   ```

2. **Run Dev Environment:**
   ```bash
   npm start
   ```

3. **Build Project:**
   ```bash
   npm run build
   ```

---

## 🌐 Deployment
This project is configured with a automated build pipeline. To deploy:
- **Netlify:** Drag and drop the `build/` directory, or link the repo with build command `npm run build` and directory `build`.
- **GitHub Pages:** Runs `npm run deploy` to compile and publish to the `gh-pages` branch.
